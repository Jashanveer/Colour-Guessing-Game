# Colour-Guessing-Game

#### Description
It is a fun game in which you have to guess the RGB color Code. It uses **CSS, JQuery and HTML**.

#### How to play?
<pre>
1. Select a mode easy or hard
    + Easy mode uses 3 colours and hard mode uses 6 colours.
</pre>
